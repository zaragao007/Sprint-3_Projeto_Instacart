# Análise de Comportamento de Compra - Instacart

Este projeto faz parte da Sprint 3 do curso de Análise de Dados da TripleTen. O objetivo foi realizar uma análise profunda dos dados de pedidos do Instacart para extrair insights sobre padrões de consumo, horários de pico e fidelidade dos usuários.

## 📊 Visualização Interativa (Dashboard)
O resultado final da análise pode ser explorado de forma interativa através do Tableau Public:
👉 **[Acesse o Dashboard Aqui](https://public.tableau.com/app/profile/marcelo.aragao/viz/Sprint3-ProjetoInstacart/Painel1)**

## 🛠️ Tecnologias Utilizadas
* **Python**: Linguagem principal para manipulação de dados.
* **Pandas**: Limpeza, tratamento e união das tabelas (merge).
* **Jupyter Notebook**: Ambiente de desenvolvimento e análise exploratória.
* **Tableau**: Criação de visualizações interativas e Dashboard.

## 📈 Principais Insights
* **Distribuição de Pedidos**: Identificação dos horários de maior volume (entre 10h e 16h).
* **Fidelização**: Análise de recorrência de pedidos com picos semanais e mensais.
* **Ranking de Produtos**: O setor de hortifrúti (*Produce*) domina o volume de vendas e recorrência.
